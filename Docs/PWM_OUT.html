PWM Output Module

Hardware-In-the-Loop Test Bench Documentation

Overview

The PWM Output module provides the Test Bench with the capability to generate precise Pulse Width Modulated (PWM) signals. This is used to simulate sensors or provide control signals to the Device Under Test (DUT). The module is built on the Zephyr RTOS shell architecture, allowing seamless integration with PC-based automation tools like Robot Framework.

Supported Channels

Channel 1

Pin: PA0 Hardware: TIM2_CH1

Channel 2

Pin: PA1 Hardware: TIM2_CH2

Zephyr Shell APIs

pwm_out set

Generates a PWM signal with a specified duty cycle using the default frequency (1000 Hz).

uart:~$ pwm_out set <channel_number> <duty_cycle_%>

Example: pwm_out set 1 50 (Sets Channel 1 to 50% duty cycle)

pwm_out freq

Generates a PWM signal with a custom frequency and duty cycle.

uart:~$ pwm_out freq <channel_number> <frequency_hz> <duty_cycle_%>

Example: pwm_out freq 2 5000 25 (Sets Channel 2 to 5 kHz at 25% duty cycle)

pwm_out stop

Safely stops the PWM signal on the specified channel by setting the pulse width to zero.

uart:~$ pwm_out stop <channel_number>

Example: pwm_out stop 1 (Stops Channel 1)
