Hardware-in-the-Loop Test Bench

A fully automated, Zephyr RTOS-powered validation suite designed for testing Electronic Control Units (ECUs) and Automotive/Industrial components.

Project Overview

This project transforms an STM32F401 Blackpill into a powerful validation tool. By bridging a Python/Robot Framework host application with an interrupt-driven Zephyr RTOS firmware, the Test Bench allows QA engineers and developers to instantly inject faults, send serial streams, and measure precisely timed hardware signals (like PWM) completely autonomously.

Continuous Integration (CI/CD)

Fully compatible with Jenkins and GitHub actions. Run thousands of physical hardware test cases automatically on every code commit.

Robot Framework Powered

Engineers can write tests in plain English using our custom TestBenchLibrary.py keywords, requiring zero embedded programming knowledge.

Sub-millisecond Precision

Utilizing hardware timers, Direct Memory Access (DMA), and Interrupt Service Routines to guarantee strict timing constraints are met.

Hardware Pinout Reference

Ensure your jumper wires and logic analyzers are connected to the correct STM32 pins before launching the Robot Framework suite.
